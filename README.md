### Requirements
- [Python 3.14](https://www.python.org/downloads/release/python-3140/) (or later)
- [Flask 3.1.x](https://pypi.org/project/Flask/)
- [Requests module 2.32.5](https://pypi.org/project/requests/) (or later)
- [Winotify module 1.1.0](https://pypi.org/project/winotify/) (or later); used for Windows toast notifications
- [geopy module 2.4.1](https://github.com/geopy/geopy) (or later)

### How to run
1. Download Python 3.14 (or later) along with all dependencies listed above.
2. Open terminal and clone this repository to a folder of your choice:
```
cd D:\folder-of-choice
git clone https://github.com/lezniakb/TrafiTracker.git
```

3. Enter the project folder
```
cd TrafiTracker
```
4. Run the application
```
flask --app main.py run
```
